\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
Sudoku (数独, sūdoku, digit-\/single) (/suːˈdoʊkuː/, /-\/ˈdɒk-\//, /sə-\//, originally called Number Place)\mbox{[}1\mbox{]} is a logic-\/based,\mbox{[}2\mbox{]}\mbox{[}3\mbox{]} combinatorial\mbox{[}4\mbox{]} number-\/placement puzzle. In classic sudoku, the objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids that compose the grid (also called \char`\"{}boxes\char`\"{}, \char`\"{}blocks\char`\"{}, or \char`\"{}regions\char`\"{}) contain all of the digits from 1 to 9. The puzzle setter provides a partially completed grid, which for a well-\/posed puzzle has a single solution.\hypertarget{index_Starting}{}\doxysection{Starting}\label{index_Starting}
